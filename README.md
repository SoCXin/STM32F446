### 关键特性

* 128 Kbytes SRAM
* USB 2.0
* 2× CAN (2.0B Active)
* 2xDAC 12bit,  3 x ADC / 24ch x 12bit 2.4MSPS(交错模式7.2MSPS)
* 20个通信接口
